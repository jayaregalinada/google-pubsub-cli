# Google PubSub CLI
Using [laravel-zero.com](https://laravel-zero.com/)

### Composer install
```sh
composer install
```

### Create Topic
```sh
php pubsub make:topic <topic> [--project_id=YOUR_PROJECT_ID]
```

### Create Subscription
```sh
php pubsub make:subscription <subscription> <topic> [--project_id=YOUR_PROJECT_ID]
```
